Что содержит программа

hebrewverbs — это офлайн-тренажёр для изучения ивритских глаголов. Программа включает:

  • 150 инфинитивов с переводами на русский, английский и украинский;
  • Таблицы спряжения по временам, родам и числам;
  • Примеры предложений с переводами и озвучкой;
  • Быстрый переключатель языков перевода;
  • Фильтры и сортировку по темам и алфавиту.

Как работать со списком инфинитивов

Раздел «Инфинитивы» содержит список всех глаголов программы. Список открывается сразу в алфавитном порядке по переводу.

  • Выберите глагол в списке слева;
  • Щёлкните по глаголу, чтобы открыть таблицу и примеры;
  • Используйте вертикальную прокрутку для просмотра всех слов;
  • При необходимости включите сортировку по ивриту или по темам.

Таблицы спряжения

Для каждого глагола представлены формы:

  • Настоящее время (м/ж, ед./мн.);
  • Прошедшее время для всех лиц;
  • Будущее время по всем формам;
  • Императив (повелительное наклонение), если доступно.

Таблица открывается автоматически после выбора глагола.

Примеры употребления

Над таблицей отображаются примеры предложений:

  • Иврит + перевод на выбранный язык;
  • Кнопка озвучки каждого примера;
  • Кнопка «Примеры/Таблица» для переключения режима;
  • Перевод можно менять в один клик (RU / EN / UA).

Рекомендации

  • Учите 3–5 глаголов в день;
  • Повторяйте вслух и используйте озвучку;
  • Сравнивайте примеры между временами;
  • Запоминайте корни и биньяны — это ускоряет обучение.
